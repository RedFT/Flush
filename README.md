Flush
=====

Little dudes in the sewer
** it's a work in progress


Requirements
=====

Python, Pygame, BeautifulSoup4


To Run
=====

Open a terminal and run

$ python main.py


Controls
=====

5       - Switch Between Players

shift   - Shoot

/       - Big Jump

up      - Jump

left    - Move Left

right   - Move right

p       - Pause
