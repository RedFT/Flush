Flush
=====

Little dudes in the sewer

![Alt text](/resources/Flush.gif?raw=true "Short Demo")


Requirements
=====

Python, Pygame, BeautifulSoup4


To Run
=====

Open a terminal and run

$ python main2.py


Controls
=====

5       - Switch Between Players

shift   - Shoot

/       - Big Jump

up      - Jump

left    - Move Left

right   - Move right

p       - Pause
